# ls

列出当前工作目录或指定目录下所含的文件及目录信息。 全拼是 list 。

用法：

ls [选项]... [目录或文件名]...

选项：

-a, --all 显示所有文件及目录包括 .开头的隐藏档案 -F, --classify 列表会加上文件类型的指示符号(* / = @ | 其中一个, 如,＊ 可执行文件, / 目录, ＠ 链接文件, | 管道文件) -h, --human-readable 与 -l 一起，以便于阅读的格式输出文件大小(如 1K 234M 2G) -l 列出更加详细信息 -R, --recursive 递归显示子目录 -S 根据文件大小排序 -t 根据修改时间排序 -r 将以相反顺序显示 --full-time 显示完整时间

示例：

[root@localhost usr]# ls
bin  etc  games  include  lib  lib64  libexec  local  sbin  share  src  tmp
[root@localhost usr]# ls /
bin   dev  home  lib64       media  mnt  opt   root  selinux  sys  usr
boot  etc  lib   lost+found  misc   net  proc  sbin  srv      tmp  var

[root@localhost usr]# ls -lh
总用量 156K
dr-xr-xr-x.   2 root root  36K 4月  10 2019 bin
drwxr-xr-x.   2 root root 4.0K 9月  23 2011 etc
drwxr-xr-x.   2 root root 4.0K 9月  23 2011 games
drwxr-xr-x.  34 root root 4.0K 1月  21 2016 include
dr-xr-xr-x.  28 root root 4.0K 4月   4 2019 lib
省略...

# cd

切换到指定工作目录。 全拼是 change directory 。

用法：

cd [目录]

其中 dir 可为绝对路径或相对路径，若目录名称省略，则变换至使用者的 home 目录。

在 linux 中 "~" 符号也表示 home 目录的意思，~account 代表 account 这个账户的 home 目录，"-" 代表上一次的工作目录，"." 则是表示目前所在的目录，".." 则表示当前目录的上一层目录。

示例：

[root@localhost /]# cd
[root@localhost ~]# 

[root@localhost /]# cd ~
[root@localhost ~]# 

[root@localhost /]# cd /usr/
[root@localhost usr]# cd ..
[root@localhost /]# 

[root@localhost /]# cd /usr/
[root@localhost usr]# cd ~
[root@localhost ~]# cd -
/usr
[root@localhost usr]#

# pwd

打印当前工作目录绝对路径名称，全拼是 print working directory

用法：

pwd [-P]

选项：

-P 打印实际物理路径(例如链接的实际路径)

示例：

[root@localhost tmp]# pwd
/usr/tmp
[root@localhost tmp]# pwd -P
/var/tmp

# mkdir

创建目录(文件夹)，全拼是 make directories

用法：

mkdir [选项]... 目录...

选项：

-p, --parents 创建目标目录及其上级，即使上级目录不存在，会按目录层级自动创建目录 -m, --mode=模式 设置目录权限(类似chmod)

示例：

[root@localhost opt]# mkdir test
[root@localhost opt]# ls
test

[root@localhost opt]# mkdir -p test1/test
[root@localhost opt]# ls test1
test

[root@localhost opt]# mkdir -m 777 test2
[root@localhost opt]# ls -l
drwxr-xr-x. 2 root root 4096 2月   1 14:14 test
drwxr-xr-x. 3 root root 4096 2月   1 14:15 test1
drwxrwxrwx. 2 root root 4096 2月   1 14:19 test2

[root@localhost opt]# mkdir test3 test4
[root@localhost opt]# ls
test  test1  test2  test3  test4

# touch

修改文件或目录的时间戳属性，若文件不存在，则建立一个新的空文件。

用法

touch [选项]... 文件...

选项：

-c, --no-create 不创建任何文件 -d, --date=字符串 使用指定字符串表示时间而非当前时间

示例：

[root@localhost opt]# touch test.txt
[root@localhost opt]# ls -l --full-time
-rw-r--r--. 1 root root    0 2020-02-03 13:41:26.359017598 +0800 test.txt

# 文件已存在，档案时间改为当前系统时间
[root@localhost opt]# touch test.txt
[root@localhost opt]# ls -l --full-time
-rw-r--r--. 1 root root    0 2020-02-03 13:43:38.405017540 +0800 test.txt

# rm

删除一个或多个文件或目录，全拼是 remove

用法

rm [选项]... 文件...

选项：

-f, --force 强制删除。忽略不存在的文件，不提示确认 -i 在删除前需要确认 -I 在删除超过三个文件或者递归删除前要求确认。此选项比-i 提示内容更少，但同样可以阻止大多数错误发生 -r, -R, --recursive 递归删除目录及其内容

# cp

复制文件或目录，全拼是 copy

用法

cp [选项]... 源文件 目标文件 cp [选项]... 源文件... 目录 cp [选项]... -t 目录 源文件...

选项：

-a, --archive 保留链接、文件的属性等 -f, --force 覆盖已经存在的目标文件而不给出提示(使用 -n 选项时该选项失效) -n, --no-clobber 不要覆盖已存在的文件 -R, -r, --recursive 递归复制目录及其子目录内的所有内容 -s, --symbolic-link 只创建符号链接而不复制文件 -t, --target-directory=目录 将所有参数指定的源文件/目录复制至目标目录
